# Smart Petrol Cost Calculator

## Project Description
Smart Petrol Cost Calculator is a mobile application developed to estimate petrol expenses for users in Malaysia. The application calculates the total petrol cost based on travel distance, fuel consumption, and fuel price. It also applies the BUDI MADANI fuel subsidy for eligible users to determine the final payable amount and total savings.

## Project Objectives
1. To develop a mobile application for estimating petrol costs in Malaysia.
2. To assist users in calculating their fuel expenses accurately.
3. To implement the BUDI MADANI fuel subsidy calculation for eligible users.
4. To provide users with a clear breakdown of payable fuel costs and savings.
   
## Features
- User-friendly mobile interface
- Petrol cost estimation
- Fuel consumption calculation
- Travel distance input
- BUDI MADANI subsidy eligibility option
- Final payable amount calculation
- Total savings calculation
- Instant result display

## Technology
- Android Studio
- Java
- XML
- Android SDK

## System Requirements
Android device or emulator

## Installation Guide
1. Clone or download the project repository.
2. Open the project using Android Studio.
3. Allow Gradle to sync all dependencies.
4. Run the application on an Android emulator or physical device.

## How to Use
1. Enter the travel distance.
2. Enter the vehicle fuel consumption.
3. Enter the current petrol price.
4. Select whether the user is eligible for the BUDI MADANI subsidy.
5. Tap the Calculate button.
6. View the total petrol cost, subsidy amount, final payable amount, and total savings.
